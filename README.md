# Aiham AlRawashdeh — Portfolio

Single-page Next.js portfolio. Static + ISR-ready, no backend.

## Stack

Next.js 15 · React 19 · TypeScript · Tailwind v4 · shadcn/ui · Framer Motion · Recharts · Biome · Vitest.

## Develop

```bash
pnpm install
pnpm dev
```

## Test

```bash
pnpm test
```

## Build

```bash
pnpm build && pnpm start
```

## Deploy

Push to GitHub. Connect the repo on https://vercel.com → import → defaults work. Set custom domain in Vercel project settings.

## Content

All content lives in `/content/*.ts` as typed modules. Update profile, projects, KPIs there; no other code changes needed.
